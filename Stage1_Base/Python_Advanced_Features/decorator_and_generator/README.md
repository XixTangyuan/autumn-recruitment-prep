##项目标题：Python 进阶学习｜闭包、装饰器、生成器实战练习
1.闭包：嵌套函数，内部函数引用外层变量，外层函数返回内层函数
	可以保存外层函数的局部变量不被清除
	关键字nonlocal，修改外层变量
	装饰器本质
2.装饰器：高阶函数+闭包函数
	不改变原有函数代码、执行规则，为其添加功能
	通用模板（*args,**kwargs)（元组类型，字典类型）可以兼容所有参数类型函数，减少报错，增强代码健壮性
	多层装饰器叠加执行，将已经被装饰过的函数看作整体在进行装饰
	类装饰器，将多个装饰器打包形成类，对想要装饰的函数添加装饰器
3.生成器：高级迭代器，可更少的占用内存
	关键字yield,定义生成器，调用时才产出数据
	适配海量日志，大型数据集读取场景
	遍历方式：for循环自动迭代（for i in gen:print(i))；手动next（）获取单行数据（print(next(gen)))#next只能对生成器对象使用

##实操踩坑记录
1.闭包报错：内层修改外层变量未加nonlocal，触发 UnboundLocalError
2.装饰器报错：内层函数未 return 原函数结果，返回 None，后续函数无法调用
3.装饰器参数转发遗漏：inner 没有接收*args,**kwargs，带参函数装饰直接报错
4.生成器语法错误：多余 while 死循环嵌套，代码结构错乱
5.next () 使用误区：只能对生成器对象使用 next，迭代出来的字符串 line 禁止使用 next ()
